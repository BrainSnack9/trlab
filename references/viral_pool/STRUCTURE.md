# 카드 단위 깊이 분석 — 후킹·전개·결론 구조

위 통계는 메타데이터 기반. 여기서는 정보성 고가치 4편을 카드 한 장씩 시각 분석해 후킹/전개/결론의 실제 작동 방식을 추출.

## 7. 4가지 정보성 viral 양식 비교

분석 결과 정보성 viral 카드뉴스는 **시각적으로 4가지 양식**으로 수렴함.

### 양식 A — 시네마틱 사진 + 흰 텍스트 (medical_newsroom · ER 20%)
```
[표지]
  풀블리드 영화/드라마 스틸 (분위기·은유)
  ┌─────────────────────────┐
  │ ● handle                ←  좌측 상단 작은 핸들 라벨 (◯ 점 + 핸들명)
  │                    [M]  ←  우상단 미니 워터마크
  │                         │
  │  영화 스틸 풀블리드        │
  │                         │
  │ 평생 진짜라고 믿어왔지만   │  ← 좌측 하단 거대 흰 헤드 2줄
  │ 사실은 잘못된 6가지 속설  │
  └─────────────────────────┘

[본문 — 항목 카드 × N개, 1:1 매핑]
  ┌─────────────────────────┐
  │   주제 시각화 풀블리드     │  ← 항목과 의미 연결되는 사진
  │                         │
  │           ⓝ            │  ← 가운데 하단 번호 원
  │  면도를 하면 털이 더      │  ← 클레임(가짜 명제) 굵은 흰글씨
  │   두껍게 난다            │
  │                         │
  │  털을 깎는다고 굵기가     │  ← 두 줄 짧은 반박 설명
  │  변하는 건 아닙니다.     │
  └─────────────────────────┘

[결론·CTA] 별도 카드 없음. 마지막 항목으로 자연 종결.
```
- 카드 수: 7 (표지 1 + 항목 6)
- 색: 어둡고 차분 (영화 스틸 톤)
- 시그니처: 우상단 ‘M’ 로고
- 강점: 권위감↑, 클릭률↑. 약점: 사진 큐레이션 비용

### 양식 B — 칼럼형 회색 박스 (diet.design_ · ER 23%)
```
[표지 = 칼럼 타이틀 카드]
  회색 배경
  ┌─────────────────────────┐
  │  [핑크 라벨 박스]         │  ← 칼럼 시리즈 명 (예: "초고도비만을 위한 ...")
  │                         │
  │  기초대사량을             │  ← 거대 흰 헤드라인
  │  결정하는 핵심은          │     5~6줄로 분할 줄바꿈
  │  근육량이 아니라          │     주장 = 헤드라인 그대로
  │  갑상선 입니다           │
  │                         │
  │       by 작가명         │  ← 손글씨 시그니처
  └─────────────────────────┘

[본문 = 텍스트-only 칼럼]
  검은 배경 + 흰 테두리 박스
  ┌─────────────────────────┐
  │ 굵은 헤드라인 1~2줄      │  ← 카드별 소주제
  │                         │
  │ 본문 단락 8~12줄         │  ← 한 카드가 거의 텍스트 풀페이지
  │ ...                     │
  │                         │
  │ (단락 분리)              │
  │                         │
  │ ...                     │
  └─────────────────────────┘

[마무리 = 다음 글 예고 + 시그니처]
  검은 배경
  ┌─────────────────────────┐
  │                         │
  │  다음 글에선 ...         │  ← 작은 흰 텍스트 한 줄
  │                         │
  │       by 작가명         │
  └─────────────────────────┘
```
- 카드 수: 9 (표지 1 + 칼럼 7 + 마무리 1)
- 색: 회색/검정 + 핑크 라벨
- 시그니처: 'by 습관디자인 김용환' 손글씨
- 강점: 정보 밀도↑, 진정성↑. 약점: 호흡 길어 dropoff 위험

### 양식 C — 셀카·라이프스타일 + 표지에만 텍스트 (dailybeauty_wellness · ER 18%)
- 표지: 모델 셀카 풀블리드 + 흰 텍스트 후크
- 본문: 텍스트 거의 없이 무드샷 모음 (정보는 캡션에 다 들어감)
- → 정보성으로 분류되지만 **시각 양식은 라이프스타일**. 모방 어려움 (캡션 의존)

### 양식 D — 흰배경 + 짤 + 마스코트 (kor.medicalmeme · 사용자 기존 양식)
- 표지: 흰배경 + 메가 키워드 + 침팬지(의대생) + DM 클러스터
- 본문: 흰배경 + 우상단 헤딩 + 짤+말풍선 + 인용 박스 ★
- 마무리: 흰배경 + 빨간 메가 결론 + 고정 CTA
- 강점: 친근감, 패러디 가능. 약점: 권위감↓

## 8. 카드 위치별 역할 — 4양식 통합 매핑

| 카드 위치 | A 시네마틱 | B 칼럼 | D 흰배경짤 | **공통 패턴** |
|---|---|---|---|---|
| **1 (후킹)** | 영화 스틸 + 주장 헤드 | 핑크 라벨 + 주장 헤드 + 작가 | 메가 키워드 + 부제 + DM | **주장/키워드 + 시각 후크 1요소** |
| **2 (도입)** | 첫 항목 시작 | 인용·도입 단락 | DM/마스코트 자기소개 | **상대 주장 정리** |
| **3~N-1 (전개)** | 항목 1:1 매핑 (번호 + 사진 + 클레임 + 반박) | 텍스트 풀페이지 칼럼 | 헤딩 + 짤+말풍선 + 인용박스 | **소주제별 분할 + 정보 밀도** |
| **N (마무리)** | 마지막 항목으로 종결 | "다음 글에선" + 시그니처 | 빨간 결론 + 고정 CTA | **작가/브랜드 시그니처** |

## 9. 시각 자산 분류 (정보성 고가치 4편 통합)

| 자산 | 사용 양식 | 효과 |
|---|---|---|
| 영화 스틸 / 드라마 캡쳐 | A | 권위·미감↑, 주제 은유 |
| 모델 셀카 / 무드샷 | C | 친밀감, 라이프스타일 정체성 |
| 짤 / 밈 / 스톡 동물 | D | 유머, 패러디 가능 |
| 텍스트-only 박스 | B | 정보 밀도, 호흡 |
| 권위 기관 로고 (FDA/NIH) | D | 신뢰도 시각화 |
| 핑크/빨강 강조 박스 | B, D | 시리즈 정체성, 결론 |
| 번호 원 (①②③) | A | 항목 시퀀스 명시 |
| 손글씨 시그니처 | B | 작가 정체성 |

## 10. 카드 수 vs 양식 — 어느 양식이 어떤 길이에 강한가

| 양식 | 적정 카드 수 | 이유 |
|---|---:|---|
| A 시네마틱 | 7~10 | 항목 1:1 매핑이라 토픽 수 = 카드 수 - 1 |
| B 칼럼 | 8~12 | 본문 호흡 길어야 진가 (단락 분리) |
| C 셀카 | 4~6 | 정보가 캡션에 있어 짧을 수 있음 |
| D 흰배경짤 | 9~12 | 본문 모듈 다양해 길이 자유 |
