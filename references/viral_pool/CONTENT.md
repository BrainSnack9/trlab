# 카드 단위 깊이 분석 — 후킹·전개·결론 카피 공식

위 통계는 캡션 톤 분석. 여기서는 카드별 시각·텍스트를 함께 본 결과 추출한 카피 패턴.

## 8. 후킹 카피 5가지 공식

표지에서 손가락을 멈추게 하는 카피는 다음 5개 공식 중 하나로 수렴.

### 공식 1 — 반전 단언 (medical_newsroom 양식)
**`평생 [믿음]지만, 사실은 [반대 결론]`**
- `평생 진짜라고 믿어왔지만 사실은 잘못된 6가지 속설` (medical_newsroom, ER 20%)
- `[명사]만 알면 충분하다고? 사실은...`
- `[명사]는 [효능]이 있다? 연구가 말하는 진짜`

→ 적용: 의학 팩트체크에 가장 강력. 토픽 1개 + 반전 1개로 명확.

### 공식 2 — 메가 키워드 + 도발 부제 (kor.medicalmeme 양식)
**`[3~5글자 키워드]` + `[반전 부제]`**
- `공복버터` + `진짜 말이 안되는 이유`
- `비타민C` + `의사들도 개같이 싸우는 이유`
- `카니보어` + `일단 나는 절대 안하는 이유`

→ 적용: 짧은 명사형 토픽이 있을 때.

### 공식 3 — 자가 약속·시리즈 라벨 (diet.design_ 양식)
**`[작가/시리즈 라벨]` + `[주장 그대로의 헤드라인]`**
- 핑크 라벨 `초고도비만을 위한 습관다이어트 칼럼` + 헤드라인 `기초대사량을 결정하는 핵심은 근육량이 아니라 갑상선 입니다`
- 시그니처 `by 습관디자인 김용환`

→ 적용: 전문가 포지션, 시리즈 운영 시. 라벨은 매번 동일.

### 공식 4 — 즉답형 후크 (dailybeauty_wellness 양식)
**`[증상] 하면 [해결]`**
- `급 야식 땡긴다면 / 이마를 때리면 돼`
- `눈 밑 다크써클 / 이거 하나면 끝`

→ 적용: 라이프 핫토픽, 즉시 검증 가능한 클레임.

### 공식 5 — 숫자 약속형 (medical_newsroom 표지의 6가지 = 이 공식)
**`[숫자] + 가지 / 개 + [카테고리]`**
- `잘못된 6가지 속설`
- `먹으면 안 되는 음식 5개`
- `의사가 추천하는 영양제 3가지`

→ 적용: 항목 카드를 1:1로 분할할 때. 표지에서 분량 약속 → 끝까지 보게 함.

## 9. 본문 전개 4가지 패턴

### 패턴 ① 열거형 (1:1 매핑) — medical_newsroom · idolissue
- 카드 = 항목 (번호 ①②③ 명시)
- 각 카드: `사진 풀블리드 + 번호 + 클레임 + 반박 2줄`
- 표지의 “6가지” 약속 ↔ 본문 6장 1:1 매핑
- **장점**: 인지 부담 적음, dropoff 적음
- **언제**: 토픽이 병렬 항목일 때 (속설 6개, 영양제 5개)

### 패턴 ② 칼럼형 (텍스트 풀페이지) — diet.design_
- 카드 = 한 단락 ~ 한 섹션
- 굵은 소제목 + 본문 8~12줄
- 줄바꿈으로 호흡 분리, 한 카드가 mini essay
- **장점**: 정보 밀도↑, 깊이↑
- **언제**: 메커니즘·인과 설명, 전문가 포지션
- **위험**: 호흡 길어 끝까지 안 봄 → 필수: 카드 1마다 mini-payoff

### 패턴 ③ 인용·반박형 (모듈 ABCD 조합) — kor.medicalmeme
- `정의 → 짤+코믹 → 인용 박스 ★ → 권위 합의 → 결론`
- 다양한 시각 자산 (짤, 권위 로고, 그래프 캡션) 카드별 변주
- **장점**: 시각 다양성, 패러디 가능
- **언제**: 가짜 의학 팩트체크 (반증 자료 풍부할 때)

### 패턴 ④ 자기서사형 — diet_day3 · su_jineee
- "안녕하세요" / "오늘은" 으로 시작
- 1인칭 화자 + 개인 경험 + 정보 혼합
- **장점**: 친밀감, 신뢰
- **위험**: 정보성보다 일상에 가까워 진정성 의존
- **언제**: 작가 정체성이 분명할 때

## 10. 결론·CTA 카피 공식

### 공식 A — 빨간 메가 결론 + 고정 CTA (kor.medicalmeme)
- 빨간 큰 글씨: `저는 적정량 섭취하겠습니다.` (1인칭 다짐형)
- 고정 카피: `거짓 의학이 판치는 세상 / 대신 공부하고 조사해드립니다.` + `팔로우하고 진짜 검증된 정보만 받아보세용`

### 공식 B — 항목 마지막으로 자연 종결 (medical_newsroom)
- 별도 결론 카드 없음. 마지막 항목(⑥)이 곧 마무리
- 캡션 마지막 줄에 출처 명시: `📖 Mayo Clinic, Harvard Health Publishing 외 의학 자료 기반 정리`
- → CTA는 시각보다 캡션·해시태그로

### 공식 C — 다음 글 예고 + 작가 시그니처 (diet.design_)
- 마지막 카드: `다음 글에선 ...` + `by 작가명`
- 시리즈 운영의 일관성, 다음편 기대 유도
- 캡션 끝에 상담 신청 ⬇️ 패턴

### 공식 D — 단순 인포 종료 (dailybeauty_wellness)
- 마지막 카드도 셀카, 별도 CTA 없음
- 캡션에 모든 정보 + 해시태그

| B 칼럼 | 90% | 10% | ~3500자 (본문 재발행) |

## 13. 후킹 카피 응용 가이드 (사용자 양식 기준)

사용자 양식(D)에서 표지 카피 만들 때:

1. 토픽 → **3~5글자 명사**로 압축 (`애사비`, `공복버터`)
2. 부제 4패턴 중 하나 선택 (`진짜 말이 안되는 이유` / `의사들도 개같이 싸우는 이유` / `일단 나는 절대 안하는 이유` / `[A]에..? [B]이랑..?`)
3. 보조: 메가키워드+숫자 약속 (`잘못된 6가지`) 추가 가능

→ 만약 양식 A로 변환할 거면: `평생 [믿음]지만, 사실은 [반대]` 공식 + 영화 스틸 풀블리드.
