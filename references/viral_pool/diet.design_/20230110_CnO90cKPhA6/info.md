## 캡션

```
기초대사를 근육량으로 올리는 건 정확한 이야기가 아니다
- 많은 분이 감량식으로 빡세게 근력운동을 함
- 목적은 근육량을 높여 기초대사량을 늘린다는 것
- 하지만 기초대사의 핵심은 근육이 아님
- 근육은 1kg당 고작 14kcal 정도의 기초대사만 차지

기초대사량의 핵심은 시상하부-뇌하수체-갑상선 축
- 우리 몸의 항상성을 담당하는 시상하부
- 몸의 변화를 감지하여 뇌하수체를 거쳐 갑상선에 명령
- 갑상선 호르몬 양에 따라 대사가 높아지가도 낮아지기도
- 열량 풍부 - 대사 올라감, 열량 부족 대사 줄어듦

기초대사량에 가장 큰 영향을 미치는 것은 먹는 양
- 적게 먹으면서 기초대사량 올리기는 정말 힘듦
- 갑상선 호르몬 줄어들면서 대사량 줄어들리 때문
- 회복법은 다시 식사량을 늘리는 것외에 딱히 없음
- 운동은 기초대사량이 아니라 '활동'대사량'을 높이는 것 

문제가 생겨서야 겨우 인식하게 되는 갑상선의 중요성
- 갑상선 인식 시기는 아쉽게도 다이어트 휴유증 겪을 때
- 무월경, 불규칙적이거나 양 많음, 탈모, 소화불량, 부종
- 갑상선 호르몬이 난소, 모발에 직간접적 영향 끼침
- 다시 식사 원래대로 해도 즉각적으로 좋아지지 않음

만성적인 갑상선 문제로 인한 후유증까지
- 갑상선 호르몬 분비 자체에 문제가 생기는 저하증 항진증
- 특히나 저하증은 살이 잘 안 빠지는 몸이 된다는 뜻
- 소위 물만 먹어도 살이 찌는 상태가 이런 상황
- 이런 분은 살이 쪄도 대사가 많이 올라가지 않는다

단기적인 강한 체중감량이 여성에게 큰 문제인 이유
- 일시적인 병원처방을 통해 몸을 안정시키기도 함
- 갑상선 기능 저하증은 남성보다 여성이 3~4배 많음
- 특히 여성의 단기적인 빠른 다이어트가 문제 촉발
- 당장의 바프 때문에 장기적 호르몬 방해를 겪을 수 있음
```
