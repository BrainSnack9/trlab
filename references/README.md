# 참고 자료

벤치마킹, 트렌드 분석, 영감이 되는 레퍼런스를 모아두는 공간입니다.

## 분류 방법

```
[날짜]_[출처/브랜드명]_[주제].md
예) 20260506_국민건강보험_건강정보_레퍼런스.md
```

## 참고 자료 기록 양식

```markdown
## 출처 정보
- 브랜드/채널명:
- URL / 링크:
- 수집일:

## 잘된 점
-

## 적용할 아이디어
-

## 스크린샷
(이미지 첨부)
```

## 트렌드 수집 채널

- 인스타그램: 팔로우할 계정 목록 관리
- 핀터레스트: 보드 링크
- 비헨스(Behance): 카드뉴스 키워드 검색
- 국내 채널: 카카오채널, 네이버 포스트 우수 사례
